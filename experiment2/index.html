Experiment 2 - Vectors/Interactivity/Animation

Imitate

I started off my experiment by asking chatGPT for code that would generate a trail of squares following the user’s mouse. The reason I chose this was because it was a cool particle effect that often appears in games as an aesthetic choice, and it also allowed me to complete the interactivity part of this assignment. The initial code that ChatGPT gave me worked, but there were a bunch of little things like color and trail length that I felt could be adjusted.

The code to print a square trail is: function draw() { background(255); // Update the trail array with the current mouse location trail.push({ x: mouseX, y: mouseY }); // Draw squares from the trail for (let i = 0; i < trail.length; i++) { let alpha = map(i, 0, trail.length, 255, 0); fill(0, 150, 200, alpha); noStroke(); rectMode(CENTER); rect(trail[i].x, trail[i].y, 20, 20); } // Limit the number of points in the trail array if (trail.length > 50) { trail.splice(0, 1); } }

Integrate

One thing that I changed was the color of the squares because I felt that the blue trail was bland. Another change that I made was making the squares shrink faster, allowing for the visual effect to be smoother than before.

let trail = []; function setup() { createCanvas(400, 400); } function draw() { background(255); // Update the trail array with the current mouse location trail.push({ x: mouseX, y: mouseY, size: 20 }); // Draw squares from the trail for (let i = 0; i < trail.length; i++) { let alpha = map(i, 0, trail.length, 255, 0); fill(0, 150, 200, alpha); noStroke(); rectMode(CENTER); rect(trail[i].x, trail[i].y, trail[i].size, trail[i].size); // Shrink the size of each square trail[i].size *= 0.95; } // Remove squares that have shrunk too much trail = trail.filter(square => square.size > 1); }
